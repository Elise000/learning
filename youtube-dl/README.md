# Youtube-dl

## Installation
Using the guide provided by the homepage:
```
$	sudo wget https://yt-dl.org/downloads/2015.11.13/youtube-dl -O /usr/local/bin/youtube-dl

$	sudo chmod a+rx /usr/local/bin/youtube-dl
```

## Update
```
$ sudo youtube-dl -U
```
