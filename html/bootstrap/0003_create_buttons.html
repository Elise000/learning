Creating all kinds of buttons

Bootstrap uses CSS classes to style buttons, regardless of tags shown in examples below. the '.btn' class is setting up button outline and shapes.

As for colors, it comes with 5 standard types:

  1. default = white with outline
  2. primary = blue
  3. success = green
  4. warning = yellow
  5. danger = red
